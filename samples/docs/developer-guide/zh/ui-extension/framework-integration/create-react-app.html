# Foxit PDF SDK for Web 示例 – 由 "create-react-app" 创建的React.js

本节介绍两个示例。一个介绍如何快速运行Foxit PDF SDK for Web包中的由"create-react-app"创建的react.js样本示例，另外一个介绍如何将Foxit PDF SDK for Web集成到由"create-react-app"创建的现有的React应用程序中。

# 快速运行由 "create-react-app" 创建的 React.js样本示例

备注：在下面的介绍中，Foxit PDF SDK for Web 的根文件夹被称为 root.

Foxit PDF SDK for Web为React应用程序提供了一个由"create-react-app"创建的样板工程。该工程位于Foxit PDF SDK for Web包中的 integrations/ 目录下。

# 概述工程结构

app
├── public
│   └── index.html
├─src/
│  ├─components/
│  │  └─PDFViewer/
│  ├─foxit-lib/
│  │    ├─...
│  ├─App.css
│  ├─App.js
│  ├─index.css
│  ├─index.js
│  ├─license-key.js
│  └──preload.js
├─.eslintignore
├─config-overrides.js
├─package.json

# 主要目录和文件描述：

文件/文件夹 描述
src/ 包含app所有的JS和CSS文件。
src/components/PDFViewer/ 包含Foxit PDF SDK for Web的初始化插件。
src/preload.js 用于预加载JS引擎。
src/license-key.js 授权密钥文件。
src/app.js 应用程序入口文件。
config-overrides.js 调整Webpack配置。
package.json 列出依赖项、版本生成信息等。

# 先决条件

  • Nodejs (opens new window) 和 npm (opens new window)
  • Foxit PDF SDK for Web (opens new window)

# 开始

  • 将 root/examples/license-key.js 拷贝到 root/integrations/create-react-app/app/src.

  • 导航到 root/integrations/create-react-app/app/，然后执行:

    npm install
    npm install -S @foxitsoftware/foxit-pdf-sdk-for-web-library 
    npm install -D copy-webpack-plugin customize-cra react-app-rewired
    npm run start

现在，一切都已设置完成。打开浏览器，导航到 http://localhost:3000/ (opens new window) 启动应用程序。

# 将Web SDK集成到由"create-react-app"创建的react app中

# 先决条件

  • Nodejs (opens new window) 和 npm (opens new window)
  • React.js created by create-react-app (opens new window)
  • Foxit PDF SDK for Web (opens new window)

# 开始

  1. 使用"create-react-app"创建一个React app：

    `npx create-react-app app`

  2. 在 app 文件夹下，更新 package.json:

    "scripts": {
    "start": "react-app-rewired --max_old_space_size=8192 start",
    "build": "react-app-rewired --max_old_space_size=8192 build",
    "test": "react-app-rewired --max_old_space_size=8192 test --env=jsdom",
    "eject": "react-app-rewired --max_old_space_size=8192 eject"
},

  3. 在 app 文件夹下，添加 config-overrides.js:

     const CopyWebpackPlugin = require("copy-webpack-plugin");
 const { override, addWebpackPlugin, addWebpackExternals} = require('customize-cra');
 const path = require("path")
 
 const libraryModulePath = path.resolve('node_modules/@foxitsoftware/foxit-pdf-sdk-for-web-library');
 const libPath = path.resolve(libraryModulePath, 'lib');
 
 module.exports = override(    
     addWebpackPlugin(
         new CopyWebpackPlugin({
             patterns: [{
                 from: libPath,
                 to: 'foxit-lib',
                 force: true
             }]
         })
     ),
     addWebpackExternals(
         'UIExtension', 
         'PDFViewCtrl'
     )
 )

  4. 在 app 文件夹下，添加 .eslintignore:

    license-key.js

  5. 在 src 文件夹下，添加 preload.js:

     import preloadJrWorker from '@foxitsoftware/foxit-pdf-sdk-for-web-library/lib/preload-jr-worker';
 import { licenseKey, licenseSN } from './license-key';
 
 const libPath = "/foxit-lib/"
 
 window.readyWorker = preloadJrWorker({
     workerPath: libPath,
     enginePath: libPath+'/jr-engine/gsdk',
     fontPath: 'http://webpdf.foxitsoftware.com/webfonts/',
     licenseSN,
     licenseKey,
 });

  6. 在 src/index.js 文件中, 导入 preload.js:

     import './preload.js'

  7. 在 src 文件夹下, 添加 components/PDFViewer/index.js:

     import React from "react";
 import * as UIExtension from '@foxitsoftware/foxit-pdf-sdk-for-web-library/lib/UIExtension.full.js';
 import "@foxitsoftware/foxit-pdf-sdk-for-web-library/lib/UIExtension.css";
 
 export default class PDFViewer extends React.Component {
     constructor() {
         super();
         this.elementRef = React.createRef();
     }
 
     render() {
         return <div className="foxit-PDF" ref={this.elementRef} />;
     }
 
     componentDidMount() {
         const element = this.elementRef.current;
         const libPath = "/foxit-lib/";
         this.pdfui = new UIExtension.PDFUI({
             viewerOptions: {
                 libPath,
                 jr: {
                     readyWorker: window.readyWorker
                 }
             },
             renderTo: element,
             appearance: UIExtension.appearances.adaptive,
             addons: UIExtension.PDFViewCtrl.DeviceInfo.isMobile ?
             libPath+'uix-addons/allInOne.mobile.js':
             libPath+'uix-addons/allInOne.js'
         });
     }
     componentWillUnmount() {
         this.pdfui.destroy();
     }
 }

  8. 更新 App.js:

    import './App.css';
import PDFViewer from './components/PDFViewer';
function App() {
  return (
    <div className="App">
      <PDFViewer></PDFViewer>
    </div>
  );
}

export default App;

  9. 安装 node_modules，然后运行：

    cd app
npm install
npm install -S @foxitsoftware/foxit-pdf-sdk-for-web-library 
npm install -D copy-webpack-plugin customize-cra react-app-rewired
npm run start

  10. 现在，一切都已设置完成。打开浏览器，导航到 http://localhost:3000/ (opens new window) 启动应用程序。
